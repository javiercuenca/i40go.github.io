IRI: http://www.purl.org/i4go/domain-task/arrowheadsystem/AHcoreSystem

Software system, that belongs to the "standard" components offered by Arrowhead framework. Arrowhead core systems can occur once in each Arrowhead Local Cloud. Arriwhead Core Systems can be distinguished in Arrowhead Mandatory Core Systems and Automation supporting systems. Mandatory core systems must exist in each Arrowhead Local Cloud, Automation supporting systems can occur in an Arrowhead Local Cloud.

ArrowHead systemc back to ToC or Class ToC

IRI: http://www.purl.org/i4go/domain-task/arrowheadsystem/AHsystem

A System is what is providing and/or consuming services (see “Arrowhead Service”). A System can be the Service Provider of one or more services and at the same time the Service Consumer of one or more services. It normally includes software executing on hardware. It may also be referred to as Component or Device. A system can be user interface display, used to control the air-conditioning within a house, but is can also be a small temperature sensor that complies with the Arrowhead Framework.
has super-classes
system
has sub-classes
ArrowHead application component c, ArrowHead core component c
